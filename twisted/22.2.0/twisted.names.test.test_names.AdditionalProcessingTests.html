def _lookupSomeRecords(self, method, soa, makeRecord, target, addresses): (source)
Perform a DNS lookup against a FileAuthority configured with records as defined by makeRecord and addresses.
Parameters
method:strThe name of the lookup method to use; for example, "lookupNameservers".
soaA Record_SOA for the zone for which the FileAuthority is authoritative.
make​RecordA one-argument callable which accepts a name and returns an IRecord provider. FileAuthority is constructed with this record. The FileAuthority is queried for a record of the resulting type with the given name.
target:bytesThe extra name which the record returned by makeRecord will be pointed at; this is the name which might require extra processing by the server so that all the available, useful information is returned. For example, this is the target of a CNAME record or the mail exchange host pointed to by an MX record.
addressesA list of records giving addresses of target.
Returns
A Deferred that fires with the result of the resolver method give by method.
