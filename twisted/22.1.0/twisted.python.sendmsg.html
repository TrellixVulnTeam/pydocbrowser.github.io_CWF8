def recvmsg(socket, maxSize=8192, cmsgSize=4096, flags=0): (source)

Receive a message on a socket.

@param socket: The socket to receive the message on. @param maxSize: The maximum number of bytes to receive from the socket using

the datagram or stream mechanism. The default maximum is 8192.
@param cmsgSize: The maximum number of bytes to receive from the socket
outside of the normal datagram or stream mechanism. The default maximum is 4096.
@param flags: Flags to affect how the message is sent. See the C{MSG_}
constants in the sendmsg(2) manual page. By default no flags are set.
@return: A named 3-tuple of the bytes received using the datagram/stream
mechanism, a L{list} of L{tuple}s giving ancillary received data, and flags as an L{int} describing the data received.
