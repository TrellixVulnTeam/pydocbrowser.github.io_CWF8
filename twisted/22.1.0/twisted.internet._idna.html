def _idnaBytes(text): (source)

Convert some text typed by a human into some ASCII bytes.

This is provided to allow us to use the U{partially-broken IDNA implementation in the standard library <http://bugs.python.org/issue17305>} if the more-correct U{idna <https://pypi.python.org/pypi/idna>} package is not available; C{service_identity} is somewhat stricter about this.

@param text: A domain name, hopefully. @type text: L{unicode}

@return: The domain name's IDNA representation, encoded as bytes. @rtype: L{bytes}
