class Shelf(DirDBM): (source)

A directory with a DBM shelf interface.

This class presents a hash-like interface to a directory of small, flat files. Keys must be strings, but values can be any given object.

Method __getitem__ C{dirdbm[foo]} Get and unpickle the contents of a file in this directory.
Method __setitem__ C{shelf[foo] = bar} Create or modify a textfile in this directory.
