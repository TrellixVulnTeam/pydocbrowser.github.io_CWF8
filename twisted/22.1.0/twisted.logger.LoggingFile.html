twisted.logger.LoggingFile

class LoggingFile: (source)

View In Hierarchy

File-like object that turns C{write()} calls into logging events.

Note that because event formats are L{str}, C{bytes} received via C{write()} are converted to C{str}, which is the opposite of what C{file} does.

@ivar softspace: Attribute to make this class more file-like under Python 2;
value is zero or one. Do not use.
