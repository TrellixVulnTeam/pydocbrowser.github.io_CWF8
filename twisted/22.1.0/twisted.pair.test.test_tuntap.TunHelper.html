def parser(self): (source)

Get a function for parsing a datagram read from a I{tun} device.

@return: A function which accepts a datagram exactly as might be read
from a I{tun} device. The datagram is expected to ultimately carry a UDP datagram. When called, it returns a L{list} of L{tuple}s. Each tuple has the UDP application data as the first element and the sender address as the second element.
