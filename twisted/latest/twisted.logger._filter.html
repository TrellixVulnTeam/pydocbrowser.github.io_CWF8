def shouldLogEvent(predicates, event): (source)

Determine whether an event should be logged, based on the result of C{predicates}.

By default, the result is C{True}; so if there are no predicates, everything will be logged.

If any predicate returns C{yes}, then we will immediately return C{True}.

If any predicate returns C{no}, then we will immediately return C{False}.

As predicates return C{maybe}, we keep calling the next predicate until we run out, at which point we return C{True}.

@param predicates: The predicates to use. @param event: An event

@return: True if the message should be forwarded on, C{False} if not.
