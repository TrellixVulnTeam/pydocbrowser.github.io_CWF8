class CommonConstructorTestsMixin: (source)

Known subclasses: twisted.names.test.test_dns.EDNSMessageConstructorTests, twisted.names.test.test_dns.MessageConstructorTests

View In Hierarchy

Tests for constructor arguments and their associated attributes that are common to both L{twisted.names.dns._EDNSMessage} and L{dns.Message}.

TestCase classes that use this mixin must provide a C{messageFactory} method which accepts any argment supported by L{dns.Message.__init__}.

TestCases must also mixin ConstructorTestsMixin which provides some custom assertions for testing constructor arguments.
