twisted.conch.test.test_userauth.FakeTransport

class FakeTransport(transport.SSHTransportBase): (source)

View In Hierarchy

L{userauth.SSHUserAuthServer} expects an SSH transport which has a factory attribute which has a portal attribute. Because the portal is important for testing authentication, we need to be able to provide an interesting portal object to the L{SSHUserAuthServer}.

In addition, we want to be able to capture any packets sent over the transport.

@ivar packets: a list of 2-tuples: (messageType, data). Each 2-tuple is
a sent packet.

@type packets: C{list} @param lostConnecion: True if loseConnection has been called on us. @type lostConnection: L{bool}
