A L{_CommandChannel} executes a command in a session channel and connects its input and output to an L{IProtocol} provider.

def _execSuccess(self, ignored): (source)

When the request to execute the command in this channel succeeds, use C{protocolFactory} to build a protocol to handle the command's input and output and connect the protocol to a transport representing those streams.

Also fire C{commandConnected} with the created protocol after it is connected to its transport.

@param ignored: The (ignored) result of the execute request
