I am a special version of a CGI script, that uses a specific executable.

This is useful for interfacing with other scripting languages that adhere to the CGI standard. My C{filter} attribute specifies what executable to run, and my C{filename} init parameter describes which script to pass to the first argument of that script.

To customize me for a particular location of a CGI interpreter, override C{filter}.

@type filter: L{str} @ivar filter: The absolute path to the executable.
