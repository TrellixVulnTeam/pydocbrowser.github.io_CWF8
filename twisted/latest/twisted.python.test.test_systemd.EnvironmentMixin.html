class EnvironmentMixin: (source)

Known subclasses: twisted.python.test.test_systemd.EnvironmentTests

View In Hierarchy

Mixin for a L{TestCase} subclass which creates a real implementation of C{inheritedDescriptors} which is based on the environment variables set by systemd. To facilitate testing, this mixin will also create a fake environment dictionary and add keys to it to make it look as if some descriptors have been inherited.
