If a custom L{WebClientConnectionCreator}-like object is passed to L{Agent.__init__} it will be used to determine the SSL parameters for HTTPS requests. When an HTTPS request is made, the hostname and port number of the request URL will be passed to the connection creator's C{creatorForNetloc} method.
