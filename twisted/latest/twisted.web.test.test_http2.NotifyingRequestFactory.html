class NotifyingRequestFactory: (source)

A L{http.Request} factory that calls L{http.Request.notifyFinish} on all L{http.Request} objects before it returns them, and squirrels the resulting L{defer.Deferred} away on the class for later use. This is done as early as possible to ensure that we always see the result.
