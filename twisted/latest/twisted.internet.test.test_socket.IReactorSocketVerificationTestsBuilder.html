class IReactorSocketVerificationTestsBuilder(ReactorBuilder): (source)

View In Hierarchy

Builder for testing IReactorSocket implementations for required methods and method signatures.

ReactorBuilder already runs IReactorSocket.providedBy to ensure that these tests will only be run on reactor classes that claim to implement IReactorSocket.

These tests ensure that reactors which claim to provide the IReactorSocket interface actually have all the required methods and that those methods have the expected number of arguments.

These tests will be skipped for reactors which do not claim to provide IReactorSocket.
