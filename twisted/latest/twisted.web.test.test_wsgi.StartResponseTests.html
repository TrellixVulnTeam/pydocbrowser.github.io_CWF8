def test_headersShouldBePlainList(self): (source)

According to PEP-3333, the headers passed to the I{start_response} callable MUST be a plain list:

The response_headers argument ... must be a Python list; i.e. type(response_headers) is ListType

However, for bug-compatibility, any sequence is accepted. In both Python 2 and Python 3, only a warning is issued when a sequence other than a list is encountered.

def test_headersShouldEachBeTuple(self): (source)

According to PEP-3333, each header passed to the I{start_response} callable should be a tuple:

The response_headers argument is a list of (header_name, header_value) tuples

However, for bug-compatibility, any 2 element sequence is also accepted. In both Python 2 and Python 3, only a warning is issued when a sequence other than a tuple is encountered.
