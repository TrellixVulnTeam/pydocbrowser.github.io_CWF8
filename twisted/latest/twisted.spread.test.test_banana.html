def encode(bananaFactory, obj): (source)

Banana encode an object using L{banana.Banana.sendEncoded}.

@param bananaFactory: A no-argument callable which will return a new,
unconnected protocol instance to use to do the encoding (this should most likely be a L{banana.Banana} instance).

@param obj: The object to encode. @type obj: Any type supported by Banana.

@return: A L{bytes} instance giving the encoded form of C{obj}.
