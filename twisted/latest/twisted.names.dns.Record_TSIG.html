A transaction signature, encapsulated in a RR, as described in U{RFC 2845 <https://tools.ietf.org/html/rfc2845>}.

@type algorithm: L{Name} @ivar algorithm: The name of the signature or MAC algorithm.

@type timeSigned: L{int} @ivar timeSigned: Signing time, as seconds from the POSIX epoch.

@type fudge: L{int} @ivar fudge: Allowable time skew, in seconds.

@type MAC: L{bytes} @ivar MAC: The message digest or signature.

@type originalID: L{int} @ivar originalID: A message ID.

@type error: L{int} @ivar error: An error code (extended C{RCODE}) carried

in exceptional cases.

@type otherData: L{bytes} @ivar otherData: Other data carried in exceptional cases.
