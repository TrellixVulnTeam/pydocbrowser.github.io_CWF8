def addValue(name, value): (source)

Add the given header name/value pair.

If the given name is L{bytes}, the value must also be L{bytes}.

If the given name is L{str}, it will be encoded as ISO-8859-1, and the value, which must also be L{str}, will be encoded as ISO-8859-1.
