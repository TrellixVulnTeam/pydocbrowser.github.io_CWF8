class StructuredVoidFormat: (source)

View In Hierarchy

Formatter for structured np.void objects.

This does not work on structured alias types like np.dtype(('i4', 'i2,i2')), as alias scalars lose their field information, and the implementation relies upon np.void.__getitem__.
