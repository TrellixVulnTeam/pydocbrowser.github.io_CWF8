def bind_expression(self, bindvalue): (source)
overrides sqlalchemy.types.TypeDecorator.bind_expression

Given a bind value (i.e. a :class:`.BindParameter` instance), return a SQL expression which will typically wrap the given parameter.

.. note:

    This method is called during the **SQL compilation** phase of a
    statement, when rendering a SQL string. It is **not** necessarily
    called against specific values, and should not be confused with the
    :meth:`_types.TypeDecorator.process_bind_param` method, which is
    the more typical method that processes the actual value passed to a
    particular parameter at statement execution time.

Subclasses of :class:`_types.TypeDecorator` can override this method to provide custom bind expression behavior for the type. This implementation will **replace** that of the underlying implementation type.
