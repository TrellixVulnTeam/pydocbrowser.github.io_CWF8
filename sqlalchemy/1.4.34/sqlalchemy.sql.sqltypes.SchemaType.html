sqlalchemy.sql.sqltypes.SchemaType

class documentation

class SchemaType(SchemaEventTarget): (source)

Known subclasses: sqlalchemy.types.Boolean, sqlalchemy.types.Enum

View In Hierarchy

Mark a type as possibly requiring schema-level DDL for usage.

Supports types that must be explicitly created/dropped (i.e. PG ENUM type) as well as types that are complimented by table or schema level constraints, triggers, and other rules.

:class:`.SchemaType` classes can also be targets for the :meth:`.DDLEvents.before_parent_attach` and :meth:`.DDLEvents.after_parent_attach` events, where the events fire off surrounding the association of the type object with a parent :class:`_schema.Column`.
