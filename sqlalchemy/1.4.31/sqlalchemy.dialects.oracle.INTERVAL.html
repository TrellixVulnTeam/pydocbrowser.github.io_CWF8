def __init__(self, day_precision=None, second_precision=None): (source)

Construct an INTERVAL.

Note that only DAY TO SECOND intervals are currently supported. This is due to a lack of support for YEAR TO MONTH intervals within available DBAPIs.

Parameters
day​_precisionthe day precision value. this is the number of digits to store for the day field. Defaults to "2"
second​_precisionthe second precision value. this is the number of digits to store for the fractional seconds field. Defaults to "6".
def as_generic(self, allow_nulltype=False): (source)
overrides sqlalchemy.types.TypeEngine.as_generic

Return an instance of the generic type corresponding to this type using heuristic rule. The method may be overridden if this heuristic rule is not sufficient.

>>> from sqlalchemy.dialects.mysql import INTEGER
>>> INTEGER(display_width=4).as_generic()
Integer()

>>> from sqlalchemy.dialects.mysql import NVARCHAR
>>> NVARCHAR(length=100).as_generic()
Unicode(length=100)
New in version 1.4.0b2.
