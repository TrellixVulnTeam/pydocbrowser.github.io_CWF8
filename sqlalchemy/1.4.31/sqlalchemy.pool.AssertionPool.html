sqlalchemy.pool.AssertionPool

class documentation

class AssertionPool(Pool): (source)

View In Hierarchy

A _pool.Pool that allows at most one checked out connection at any given time.

This will raise an exception if more than one connection is checked out at a time. Useful for debugging code that is using more connections than desired.
