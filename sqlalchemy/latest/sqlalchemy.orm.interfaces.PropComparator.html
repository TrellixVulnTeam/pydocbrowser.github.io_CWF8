def and_(self, *criteria): (source)
overridden in sqlalchemy.orm.attributes.QueryableAttribute, sqlalchemy.orm.properties.RelationshipProperty.Comparator

Add additional criteria to the ON clause that's represented by this relationship attribute.

E.g.:

    stmt = select(User).join(
        User.addresses.and_(Address.email_address != 'foo')
    )

    stmt = select(User).options(
        joinedload(User.addresses.and_(Address.email_address != 'foo'))
    )
