sqlalchemy.engine.result.MappingResult

class documentation

class MappingResult(_WithKeys, FilterResult):

View In Hierarchy

A wrapper for a _engine.Result that returns dictionary values rather than _engine.Row values.

The _engine.MappingResult object is acquired by calling the _engine.Result.mappings method.
