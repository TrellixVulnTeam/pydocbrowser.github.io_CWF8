class DropboxTeam(_DropboxTransport, DropboxTeamBase): (source)

View In Hierarchy

Use this class to make requests to the Dropbox API using a team's access token. Methods of this class are meant to act on the team, but there is also an :meth:`as_user` method for assuming a team member's identity.
Method as​_admin Allows a team credential to assume the identity of an administrator on the team and perform operations on any team-owned content.
Method as​_user Allows a team credential to assume the identity of a member of the team.
