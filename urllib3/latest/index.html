def connection_from_url(url, **kw): (source)

Given a url, return an .ConnectionPool instance of its host.

This is a shortcut for not having to parse out the scheme, host, and port of the url before creating an .ConnectionPool instance.

Example:

>>> conn = connection_from_url('http://google.com/')
>>> r = conn.request('GET', '/')
Parameters
urlAbsolute URL string that must include the scheme. Port is optional.
**kwPasses additional parameters to the constructor of the appropriate .ConnectionPool. Useful for specifying things like timeout, maxsize, headers, etc.
