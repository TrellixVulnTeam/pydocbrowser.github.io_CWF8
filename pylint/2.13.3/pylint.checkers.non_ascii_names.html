pylint.checkers.non_ascii_names

module documentation
(source)

All alphanumeric unicode character are allowed in Python but due to similarities in how they look they can be confused.

See: https://www.python.org/dev/peps/pep-0672/#confusable-characters-in-identifiers

The following checkers are intended to make users are aware of these issues.
