def assertAddsMessages(self, *messages, ignore_position=False): (source)

Assert that exactly the given method adds the given messages.

The list of messages must exactly match *all* the messages added by the method. Additionally, we check to see whether the args in each message can actually be substituted into the message string.

Using the keyword argument `ignore_position`, all checks for position arguments (line, col_offset, ...) will be skipped. This can be used to just test messages for the correct node.
