def _is_attribute_property(name, klass): (source)

Check if the given attribute *name* is a property in the given *klass*.

It will look for `property` calls or for functions with the given name, decorated by `property` or `property` subclasses. Returns ``True`` if the name is a property in the given klass, ``False`` otherwise.
