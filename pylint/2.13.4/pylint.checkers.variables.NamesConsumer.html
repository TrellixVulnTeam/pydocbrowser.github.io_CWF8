def get_next_to_consume(self, node): (source)

Return a list of the nodes that define `node` from this scope.

If it is uncertain whether a node will be consumed, such as for statements in except blocks, add it to self.consumed_uncertain instead of returning it. Return None to indicate a special case that needs to be handled by the caller.
