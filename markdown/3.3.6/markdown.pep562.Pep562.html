Backport of PEP 562 <https://pypi.org/search/?q=pep562>.

Wraps the module in a class that exposes the mechanics to override `__dir__` and `__getattr__`. The given module will be searched for overrides of `__dir__` and `__getattr__` and use them when needed.

Method __dir__ Return the overridden `dir` if one was provided, else apply `dir` to the module.
Method __getattr__ Attempt to retrieve the attribute from the module, and if missing, use the overridden function if present.
Method __init__ Acquire `__getattr__` and `__dir__`, but only replace module for versions less than Python 3.7.
