django.db.migrations.operations.RunSQL

class documentation

class RunSQL(Operation): (source)

View In Hierarchy

Run some raw SQL. A reverse SQL statement may be provided.

Also accept a list of operations that represent the state change effected by this SQL change, in case it's custom column/table creation/deletion.
