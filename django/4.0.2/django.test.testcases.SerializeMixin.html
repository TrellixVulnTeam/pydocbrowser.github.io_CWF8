class SerializeMixin: (source)

View In Hierarchy

Enforce serialization of TestCases that share a common resource.

Define a common 'lockfile' for each set of TestCases to serialize. This file must exist on the filesystem.

Place it early in the MRO in order to isolate setUpClass()/tearDownClass().
