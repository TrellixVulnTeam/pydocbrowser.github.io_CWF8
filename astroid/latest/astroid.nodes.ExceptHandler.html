astroid.nodes.ExceptHandler

class documentation

class ExceptHandler(mixins.MultiLineBlockMixin, mixins.AssignTypeMixin, Statement): (source)

View In Hierarchy

Class representing an ast.ExceptHandler. node.

An ExceptHandler is an except block on a try-except.

>>> import astroid
>>> node = astroid.extract_node('''
    try:
        do_something()
    except Exception as error:
        print("Error!")
    ''')
>>> node
<TryExcept l.2 at 0x7f23b2e9d908>
>>> node.handlers
[<ExceptHandler l.4 at 0x7f23b2e9e860>]
