class safe_reraise(object):

View In Hierarchy

Reraise an exception after invoking some handler code.

Stores the existing exception info before invoking so that it is maintained across a potential coroutine context switch.

e.g.:

try:
    sess.commit()
except:
    with safe_reraise():
        sess.rollback()
