class StatementError(SQLAlchemyError): (source)

Known subclasses: sqlalchemy.exc.DBAPIError

View In Hierarchy

An error occurred during execution of a SQL statement.

:class:`StatementError` wraps the exception raised during execution, and features :attr:`.statement` and :attr:`.params` attributes which supply context regarding the specifics of the statement which had an issue.

The wrapped exception object is available in the :attr:`.orig` attribute.
