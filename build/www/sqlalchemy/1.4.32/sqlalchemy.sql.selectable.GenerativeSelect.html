sqlalchemy.sql.selectable.GenerativeSelect

class GenerativeSelect(DeprecatedSelectBaseGenerations, SelectBase): (source)

Known subclasses: sqlalchemy.sql.expression.CompoundSelect, sqlalchemy.sql.expression.Select

View In Hierarchy

Base class for SELECT statements where additional elements can be added.

This serves as the base for :class:`_expression.Select` and :class:`_expression.CompoundSelect` where elements such as ORDER BY, GROUP BY can be added and column rendering can be controlled. Compare to :class:`_expression.TextualSelect`, which, while it subclasses :class:`_expression.SelectBase` and is also a SELECT construct, represents a fixed textual string which cannot be altered at this level, only wrapped as a subquery.
