Record HTTP requests made with an agent

This is suitable for low-level testing of wrapper agents. It validates the parameters of each call to IAgent.request (synchronously raising TypeError) and captures them as a RequestRecord, which can then be used to inspect the request or generate a response by firing the ~RequestRecord.deferred.

Returns

A two-tuple of:

  • An twisted.web.iweb.IAgent
  • A list of calls made to the agent's ~twisted.web.iweb.IAgent.request() method
