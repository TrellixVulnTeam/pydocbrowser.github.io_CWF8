werkzeug.datastructures.HeaderSet

class documentation

class HeaderSet(MutableSet): (source)

View In Hierarchy

Similar to the :class:`ETags` class this implements a set-like structure. Unlike :class:`ETags` this is case insensitive and used for vary, allow, and content-language headers.

If not constructed using the :func:`parse_set_header` function the instantiation works like this:

>>> hs = HeaderSet(['foo', 'bar', 'baz'])
>>> hs
HeaderSet(['foo', 'bar', 'baz'])
