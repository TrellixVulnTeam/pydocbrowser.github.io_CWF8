astroid.nodes.scoped_nodes.AsyncFunctionDef

class documentation

class AsyncFunctionDef(FunctionDef): (source)

View In Hierarchy

Class representing an :class:`ast.FunctionDef` node.

A :class:`AsyncFunctionDef` is an asynchronous function created with the `async` keyword.

>>> import astroid
>>> node = astroid.extract_node('''
async def func(things):
    async for thing in things:
        print(thing)
''')
>>> node
<AsyncFunctionDef.func l.2 at 0x7f23b2e416d8>
>>> node.body[0]
<AsyncFor l.3 at 0x7f23b2e417b8>
