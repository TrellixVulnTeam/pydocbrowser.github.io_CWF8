astroid.transforms.TransformVisitor

class documentation

class TransformVisitor: (source)

View In Hierarchy

A visitor for handling transforms.

The standard approach of using it is to call ~visit with an astroid module and the class will take care of the rest, walking the tree and running the transforms for each encountered node.
