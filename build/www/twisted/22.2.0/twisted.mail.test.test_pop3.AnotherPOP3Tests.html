def run_PASS(self, real_user, real_password, tried_user=None, tried_password=None, after_auth_input=[], after_auth_output=[]): (source)

Test a login with PASS.

If real_user matches tried_user and real_password matches tried_password, a successful login will be expected. Otherwise an unsuccessful login will be expected.

Parameters
