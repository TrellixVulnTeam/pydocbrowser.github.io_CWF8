def test_prefixedTags(self): (source)

XML elements with a prefixed name as per upper level tag definition have a start-tag of C{"<prefix:tag>"} and an end-tag of C{"</prefix:tag>"}.

Refer to U{http://www.w3.org/TR/xml-names/#ns-using} for details.

def test_prefixPropagation(self): (source)

Children of prefixed tags respect the default namespace at the point where they are rendered. Specifically, they are not influenced by the prefix of their parent as that prefix has no bearing on them.

See U{http://www.w3.org/TR/xml-names/#scoping} for details.

To further clarify the matter, the following:

<root xmlns="http://example.com/ns/test">
    <mytag xmlns="http://example.com/ns/mytags">
        <mysubtag xmlns="http://example.com/ns/mytags">
            <element xmlns="http://example.com/ns/test"></element>
        </mysubtag>
    </mytag>
</root>

Should become this after all the namespace declarations have been I{moved up}:

<root xmlns="http://example.com/ns/test"
      xmlns:mytags="http://example.com/ns/mytags">
    <mytags:mytag>
        <mytags:mysubtag>
            <element></element>
        </mytags:mysubtag>
    </mytags:mytag>
</root>
