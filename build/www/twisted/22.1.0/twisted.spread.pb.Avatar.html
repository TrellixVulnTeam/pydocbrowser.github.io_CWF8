A default IPerspective implementor.

This class is intended to be subclassed, and a realm should return an instance of such a subclass when IPerspective is requested of it.

A peer requesting a perspective will receive only a L{RemoteReference} to a pb.Avatar. When a method is called on that L{RemoteReference}, it will translate to a method on the remote perspective named 'perspective_methodname'. (For more information on invoking methods on other objects, see L{flavors.ViewPoint}.)
