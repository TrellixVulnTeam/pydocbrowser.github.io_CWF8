class Address: (source)

View In Hierarchy

Parse and hold an RFC 2821 address.

Source routes are stipped and ignored, UUCP-style bang-paths and %-style routing are not parsed.

@type domain: C{bytes} @ivar domain: The domain within which this address resides.

@type local: C{bytes} @ivar local: The local ("user") portion of this address.
