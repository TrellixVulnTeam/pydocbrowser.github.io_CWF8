View In Hierarchy

A credential factory defines a way to generate a particular kind of authentication challenge and a way to interpret the responses to these challenges. It creates L{ICredentials<twisted.cred.credentials.ICredentials>} providers from responses. These objects will be used with L{twisted.cred} to authenticate an authorize requests.
