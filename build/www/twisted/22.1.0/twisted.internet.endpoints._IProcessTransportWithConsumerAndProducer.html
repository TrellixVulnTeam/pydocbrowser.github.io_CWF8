class _IProcessTransportWithConsumerAndProducer(interfaces.IProcessTransport, interfaces.IConsumer, interfaces.IPushProducer): (source)

View In Hierarchy

An L{_IProcessTransportWithConsumerAndProducer} combines various interfaces to work around the issue that L{interfaces.IProcessTransport} is incompletely defined and doesn't specify flow-control interfaces, and that L{proxyForInterface} doesn't allow for multiple interfaces.
