def test_namespaceReuse(self): (source)

Test that reuse of namespaces does affect an element's serialization.

When one element uses a prefix for a certain namespace, this is stored in the C{localPrefixes} attribute of the element. We want to make sure that elements created after such use, won't have this prefix end up in their C{localPrefixes} attribute, too.
