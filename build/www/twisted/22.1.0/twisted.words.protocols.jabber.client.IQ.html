class IQ(domish.Element): (source)

Wrapper for a Info/Query packet.

This provides the necessary functionality to send IQs and get notified when a result comes back. It's a subclass from L{domish.Element}, so you can use the standard DOM manipulation calls to add data to the outbound request.
