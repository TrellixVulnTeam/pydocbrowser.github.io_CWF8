def createFakeSphinxProject(self): (source)

Create a fake Sphinx project for test purposes.

Creates a fake Sphinx project with the absolute minimum of source files. This includes a single source file ('index.rst') and the smallest 'conf.py' file possible in order to find that source file.

@param fileName: The last path segment of a file which may exist within

@type fileName: L{str}

@raise FailTest: If C{fileDir.child(fileName)}:
