def _formatChildException(self, exception): (source)

Format the C{exception} in preparation for writing to the status pipe. This does the right thing on Python 2 if the exception's message is Unicode, and in all cases limits the length of the message afte* encoding to 100 bytes.

This means the returned message may be truncated in the middle of a unicode escape.

@type exception: L{Exception} @param exception: The exception to format.

@return: The formatted message, suitable for writing to the
status pipe.

@rtype: L{bytes}
