An interface for querying capabilities of a POP3 server.

Any cap_* method may raise L{NotImplementedError} if the particular capability is not supported. If L{cap_EXPIRE()} does not raise L{NotImplementedError}, L{perUserExpiration()} must be implemented, otherwise they are optional. If L{cap_LOGIN_DELAY()} is implemented, L{perUserLoginDelay()} must be implemented, otherwise they are optional.
