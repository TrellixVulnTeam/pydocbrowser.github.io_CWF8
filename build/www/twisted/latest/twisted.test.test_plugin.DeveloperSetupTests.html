def test_newPluginsOnReadOnlyPath(self): (source)

Verify that a failure to write the dropin.cache file on a read-only path will not affect the list of plugins returned.

Note: this test should pass on both Linux and Windows, but may not provide useful coverage on Windows due to the different meaning of "read-only directory".
