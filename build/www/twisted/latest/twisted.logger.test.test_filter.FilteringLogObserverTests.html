def filterWith(self, filters, other=False): (source)

Apply a set of pre-defined filters on a known set of events and return the filtered list of event numbers.

The pre-defined events are four events with a count attribute set to 0, 1, 2, and 3.

Parameters
filters:Iterable[str]

names of the filters to apply. Options are:

  • "twoMinus" (count <=2),
  • "twoPlus" (count >= 2),
  • "notTwo" (count != 2),
  • "no" (False).
other:boolWhether to return a list of filtered events as well.
Returns
Union[List[int], Tuple[List[int], List[int]]]event numbers or 2-tuple of lists of event numbers.
