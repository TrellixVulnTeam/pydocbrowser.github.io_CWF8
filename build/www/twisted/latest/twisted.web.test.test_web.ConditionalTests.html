def test_modified(self): (source)

def test_unmodified(self): (source)
If a request is made with an I{If-Modified-Since} header value with a timestamp indicating a time after the last modification of the request resource, a 304 response is returned along with an empty response body and no Content-Type header if the application does not set one.
def test_unmodifiedWithContentType(self): (source)

Similar to L{test_etagMatched}, but the response should include a I{Content-Type} header if the application explicitly sets one.

This I{Content-Type} header SHOULD NOT be present according to RFC 2616, section 10.3.5. It will only be present if the application explicitly sets it.
