class TooLong(AmpError): (source)

View In Hierarchy

One of the protocol's length limitations was violated.

@ivar isKey: true if the string being encoded in a key position, false if it was in a value position.

@ivar isLocal: Was the string encoded locally, or received too long from the network? (It's only physically possible to encode "too long" values on the network for keys.)

@ivar value: The string that was too long.

@ivar keyName: If the string being encoded was in a value position, what key was it being encoded for?
