class IReactorSocketVerificationTestsBuilder(ReactorBuilder): (source)

Builder for testing L{IReactorSocket} implementations for required methods and method signatures.

L{ReactorBuilder} already runs L{IReactorSocket.providedBy} to ensure that these tests will only be run on reactor classes that claim to implement L{IReactorSocket}.

These tests ensure that reactors which claim to provide the L{IReactorSocket} interface actually have all the required methods and that those methods have the expected number of arguments.

These tests will be skipped for reactors which do not claim to provide L{IReactorSocket}.
