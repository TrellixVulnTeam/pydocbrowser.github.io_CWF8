class OurServerCmdLineClientTests(CFTPClientTestBase): (source)

View In Hierarchy

Functional tests which launch a SFTP server over TCP on localhost and check cftp command line interface using a spawned process.

Due to the spawned process you can not add a debugger breakpoint for the client code.
