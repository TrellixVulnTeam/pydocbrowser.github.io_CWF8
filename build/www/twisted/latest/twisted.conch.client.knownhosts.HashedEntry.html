A L{HashedEntry} is a representation of an entry in a known_hosts file where the hostname has been hashed and salted.

@ivar _hostSalt: the salt to combine with a hostname for hashing.

@ivar _hostHash: the hashed representation of the hostname.

@cvar MAGIC: the 'hash magic' string used to identify a hashed line in a known_hosts file as opposed to a plaintext one.
