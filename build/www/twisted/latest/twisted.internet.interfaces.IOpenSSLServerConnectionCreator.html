View In Hierarchy

A provider of IOpenSSLServerConnectionCreator can create OpenSSL.SSL.Connection objects for TLS servers.
See Also
twisted.internet.ssl
Note
Creating OpenSSL connection objects is subtle, error-prone, and security-critical. Before implementing this interface yourself, consider using twisted.internet.ssl.CertificateOptions as your contextFactory. (For historical reasons, that class does not actually implement this interface; nevertheless it is usable in all Twisted APIs which require a provider of this interface.)
