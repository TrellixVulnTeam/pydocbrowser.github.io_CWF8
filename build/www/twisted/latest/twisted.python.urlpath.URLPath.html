class URLPath: (source)

View In Hierarchy

A representation of a URL.

@ivar scheme: The scheme of the URL (e.g. 'http'). @type scheme: L{bytes}

@ivar netloc: The network location ("host"). @type netloc: L{bytes}

@ivar path: The path on the network location. @type path: L{bytes}

@ivar query: The query argument (the portion after ? in the URL). @type query: L{bytes}

@ivar fragment: The page fragment (the portion after # in the URL). @type fragment: L{bytes}
