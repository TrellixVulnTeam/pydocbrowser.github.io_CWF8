def extractRecord(resolver, name, answers, level=10): (source)

Resolve a name to an IP address, following I{CNAME} records and I{NS} referrals recursively.

This is an implementation detail of L{ResolverBase.getHostByName}.

@param resolver: The resolver to use for the next query (unless handling an I{NS} referral). @type resolver: L{IResolver}

@param name: The name being looked up. @type name: L{dns.Name}

@param answers: All of the records returned by the previous query (answers, authority, and additional concatenated). @type answers: L{list} of L{dns.RRHeader}

@param level: Remaining recursion budget. This is decremented at each recursion. The query returns L{None} when it reaches 0. @type level: L{int}

@returns: The first IPv4 or IPv6 address (as a dotted quad or colon quibbles), or L{None} when no result is found. @rtype: native L{str} or L{None}
