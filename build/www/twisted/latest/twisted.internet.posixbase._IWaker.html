Interface to wake up the event loop based on the self-pipe trick.

The U{I{self-pipe trick}<http://cr.yp.to/docs/selfpipe.html>}, used to wake up the main loop from another thread or a signal handler. This is why we have wakeUp together with doRead

This is used by threads or signals to wake up the event loop.
