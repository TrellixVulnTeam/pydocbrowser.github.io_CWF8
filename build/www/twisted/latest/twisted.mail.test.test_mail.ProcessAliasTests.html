def test_aliasResolution(self): (source)

Check that the C{resolve} method of alias processors produce the correct set of objects:

  • direct alias with L{mail.alias.AddressAlias} if a simple input is passed
  • aliases in a file with L{mail.alias.FileWrapper} if an input in the format '/file' is given
  • aliases resulting of a process call wrapped by L{mail.alias.MessageWrapper} if the format is '|process'
