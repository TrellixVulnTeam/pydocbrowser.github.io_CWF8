def requestAvatarId(credentials): (source)

Validate credentials and produce an avatar ID.

@param credentials: something which implements one of the interfaces in C{credentialInterfaces}.

@return: a L{Deferred} which will fire with a L{bytes} that identifies an avatar, an empty tuple to specify an authenticated anonymous user (provided as L{twisted.cred.checkers.ANONYMOUS}) or fail with L{UnauthorizedLogin}. Alternatively, return the result itself.

@see: L{twisted.cred.credentials}
