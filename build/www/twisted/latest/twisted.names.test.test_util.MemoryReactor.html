An L{IReactorTime} and L{IReactorUDP} provider.

Time is controlled deterministically via the base class, L{Clock}. UDP is handled in-memory by connecting protocols to instances of L{MemoryDatagramTransport}.

@ivar udpPorts: A C{dict} mapping port numbers to instances of
L{MemoryDatagramTransport}.
