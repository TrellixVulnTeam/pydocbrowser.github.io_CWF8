def loadSession(identifier, isConfidential, authenticatedBy): (source)

Load a session given the given identifier and security properties.

As an optimization for session stores where the back-end can generate session identifiers when the presented one is not found in the same round-trip to a data store, this method may return a L{Session} object with an C{identifier} attribute that does not match C{identifier}. However, please keep in mind when implementing L{ISessionStore} that this behavior is only necessary for requests where C{authenticatedBy} is L{SessionMechanism.Cookie}; an unauthenticated L{SessionMechanism.Header} session is from an API client and its session should be valid already.

@return: an existing session with the given identifier. @rtype: L{Deferred} firing with L{ISession} or failing with

L{NoSuchSession}.
