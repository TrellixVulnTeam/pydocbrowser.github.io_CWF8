class Admonitions(Transform): (source)

View In Hierarchy

Transform specific admonitions, like this:

<note>
<paragraph>
Note contents ...

into generic admonitions, like this:

<admonition classes="note">
    <title>
        Note
    <paragraph>
        Note contents ...

The admonition title is localized.
