flask.json.tag.TaggedJSONSerializer

class documentation

class TaggedJSONSerializer: (source)

View In Hierarchy

Serializer that uses a tag system to compactly represent objects that are not JSON types. Passed as the intermediate serializer to :class:`itsdangerous.Serializer`.

The following extra types are supported:

* :class:`dict` * :class:`tuple` * :class:`bytes` * :class:`~markupsafe.Markup` * :class:`~uuid.UUID` * :class:`~datetime.datetime`
