distutils.msvc9compiler

Contains MSVCCompiler, an implementation of the abstract CCompiler class for the Microsoft Visual Studio 2008.

The module is compatible with VS 2005 and VS 2008. You can find legacy support for older versions of VS in distutils.msvccompiler.
