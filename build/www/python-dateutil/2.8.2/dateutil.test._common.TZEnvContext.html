class TZEnvContext(TZContextBase):

View In Hierarchy

Context manager that temporarily sets the TZ variable (for use on nix-like systems). Because the effect is local to the shell anyway, this will apply *unless a guard is set.

If you do not want the TZ environment variable set, you may set the DATEUTIL_MAY_NOT_CHANGE_TZ_VAR variable to a truthy value.
