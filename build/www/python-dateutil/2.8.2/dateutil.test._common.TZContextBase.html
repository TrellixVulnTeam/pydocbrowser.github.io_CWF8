dateutil.test._common.TZContextBase

class documentation

class TZContextBase(object):

Known subclasses: dateutil.test._common.TZEnvContext, dateutil.test._common.TZWinContext

View In Hierarchy

Base class for a context manager which allows changing of time zones.

Subclasses may define a guard variable to either block or or allow time zone changes by redefining _guard_var_name and _guard_allows_change. The default is that the guard variable must be affirmatively set.

Subclasses must define get_current_tz and set_current_tz.
