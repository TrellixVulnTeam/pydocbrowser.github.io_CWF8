class Request(RequestHooksMixin): (source)

A user-created Request object.

Used to prepare a PreparedRequest, which is sent to the server.

Usage:

>>> import requests
>>> req = requests.Request('GET', 'https://httpbin.org/get')
>>> req.prepare()
