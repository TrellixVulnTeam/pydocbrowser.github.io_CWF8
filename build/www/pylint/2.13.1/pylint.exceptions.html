Exception classes raised by various operations within pylint.
Class ​Empty​Report​Error Raised when a report is empty and so should not be displayed.
Class ​Invalid​Args​Error Raised when passed arguments are invalid, e.g., have the wrong length.
Class ​Invalid​Message​Error Raised when a message creation, registration or addition is rejected.
Class ​Invalid​Reporter​Error Raised when selected reporter is invalid (e.g. not found).
Class ​No​Line​Supplied​Error Raised when trying to disable a message on a next line without supplying a line number.
Class ​Unknown​Message​Error Raised when an unregistered message id is encountered.
