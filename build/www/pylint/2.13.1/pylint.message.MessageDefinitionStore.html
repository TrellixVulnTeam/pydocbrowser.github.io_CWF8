def get_message_definitions(self, msgid_or_symbol): (source)

Returns the Message definition for either a numeric or symbolic id.

The cache has no limit as its size will likely stay minimal. For each message we store about 1000 characters, so even if we would have 1000 messages the cache would only take up ~= 1 Mb.
