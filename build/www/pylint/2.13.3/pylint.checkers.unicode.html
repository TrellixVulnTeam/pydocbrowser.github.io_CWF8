pylint.checkers.unicode

module documentation
(source)

Unicode and some other ASCII characters can be used to create programs that run much different compared to what a human reader would expect from them.

PEP 672 lists some examples. See: https://www.python.org/dev/peps/pep-0672/

The following checkers are intended to make users are aware of these issues.
