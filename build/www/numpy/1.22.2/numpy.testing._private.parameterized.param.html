numpy.testing._private.parameterized.param

class param(_param): (source)

View In Hierarchy

Represents a single parameter to a test case.

For example:

>>> p = param("foo", bar=16)
>>> p
param("foo", bar=16)
>>> p.args
('foo', )
>>> p.kwargs
{'bar': 16}

Intended to be used as an argument to @parameterized:

@parameterized([
    param("foo", bar=16),
])
def test_stuff(foo, bar=16):
    pass
