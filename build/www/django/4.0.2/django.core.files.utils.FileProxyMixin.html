django.core.files.utils.FileProxyMixin

class documentation

class FileProxyMixin: (source)

Known subclasses: django.core.files.File, django.core.files.temp.TemporaryFile

View In Hierarchy

A mixin class used to forward file methods to an underlaying file object. The internal file object has to be called "file":

    class FileProxy(FileProxyMixin):
        def __init__(self, file):
            self.file = file
