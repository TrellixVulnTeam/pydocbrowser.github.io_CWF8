class _SynchronousProducer: (source)

Implements interfaces: twisted.web.iweb.IBodyProducer

View In Hierarchy

A partial implementation of an IBodyProducer which produces its entire payload immediately. There is no way to access to an instance of this object from RequestTraversalAgent or StubTreq, or even a Resource: passed to `StubTreq.

This does not implement the IBodyProducer.stopProducing method, because that is very difficult to trigger. (The request from RequestTraversalAgent would have to be canceled while it is still in the transmitting state), and the intent is to use RequestTraversalAgent to make synchronous requests.
