class CodeToAst(object):

View In Hierarchy

Given a module, or a function that was compiled as part of a module, re-compile the module into an AST and extract the sub-AST for the function. Allow caching to reduce number of compiles.

Also contains static helper utility functions to look for python files, to parse python files, and to extract the file/line information from a code object.
